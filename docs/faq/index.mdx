## Do I need to change my SDK?

No. The gateway speaks the OpenAI API format. Set the base URL to the gateway and keep your client code.

## Does it support multiple providers?

Yes. The gateway integrates with many providers and can route across them.

## Can I run it locally?

Yes. Use Docker or build from source and run on `localhost:8080`.
