The gateway exposes an OpenAI compatible API surface so you can reuse existing clients.

## Base URL

Use the gateway host as the base URL, for example:

```
http://localhost:8080
```

## Common endpoints

- `/v1/chat/completions`
- `/v1/embeddings`
- `/v1/images`
- `/v1/audio`
- `/v1/files`

Non chat endpoints are proxied to the selected provider while preserving payloads.
