Ferro Labs AI Gateway is a high performance control plane for AI traffic. It provides a single OpenAI compatible API to route, secure, and observe requests across many model providers.

- [Observability](/guides/observability)
- [API overview](/api-reference/overview)

## What you can do

- Route traffic across providers with fallback, weighted, and conditional strategies.
- Keep existing OpenAI SDK usage and change only the base URL.
- Collect structured logs, metrics, and health checks out of the box.
- Add request level controls with plugins for guardrails and rate limits.
